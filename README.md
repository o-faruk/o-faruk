## `> ls ./projects --active`

**[mcp-x-ray](https://github.com/o-faruk/mcp-x-ray)** ‚Äî MCP Server Security Scanner
> Scans MCP servers for security issues by combining static analysis with dynamic sandboxing (Docker + strace) to catch what code review alone misses, then runs a Claude-based verification layer to cut down false positives before anything reaches a human.

`Python` `Docker` `strace` `Anthropic Claude` `Static Analysis`

---

**[Chronos](https://github.com/o-faruk/Chronos)** ‚Äî Orbital Conjunction Screening
> Screens satellite orbits for close-approach conjunctions using SGP4 propagation, running on a serverless AWS pipeline for scheduled screening passes.

`Python` `SGP4` `AWS Lambda` `EventBridge` `DynamoDB`

---

**[Prometheus](https://github.com/o-faruk/Prometheus)** ‚Äî Real-Time Grid Stress Forecasting
> Live grid monitoring across three U.S. regions (CAISO, ERCOT, PJM). Ingests EIA + weather data hourly and forecasts demand with LightGBM, beating EIA's own forecast in 2 of 3 regions ‚Äî served via a live API and dashboard.

`Python` `FastAPI` `LightGBM` `Prophet` `TimescaleDB` `Docker` `React` `Caddy`
